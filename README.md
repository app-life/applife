# AppLife

# アプリケーションの概要
- iOS、macOSのアプリ紹介サイト

# アプリケーションの機能一覧
- 管理者の会員登録機能/ログイン機能（gem:deviseを使用）
- 記事作成機能
- 記事作成時画像をドラッグで投稿できる機能(javascript)
- 各記事に対してコメントする機能
- コメント承認機能(trueとfalseを切り替え）
- あいまい検索機能（gem:ramsackを使用）
- PVランキング、閲覧数機能(gem:impressionistを使用)

# アプリケーション内で使用した技術
- インフラ AWS（EC2、RDS、）
- データベース MySQL
